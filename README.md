## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/Ninja-XP/DataVisualizer.git
    cd DataVisualizer
    ```
2. Install dependencies:
    ```bash
    bash scripts/install_dependencies.sh
    ```
3. Run the tool:
    ```bash
    bash scripts/run_visualizer.sh
    ```
4. Update the tool:
    ```bash
    bash scripts/update_tool.sh
    ```
